# media

Media skills: image, audio, and video processing workflows.

## Skills

| Skill | Description |
|-------|-------------|
| **epub** | Convert markdown documents and chat summaries into formatted EPUB ebook files |
| **image-enhancer** | Improves the quality of images, especially screenshots, by enhancing resolution, sharpness, and clarity |
| **imagen** | Generate images using Google Gemini's image generation capabilities |
| **slack-gif-creator** | Create animated GIFs optimized for Slack with constraints, validation tools, and animation concepts |
| **typefully** | Create, schedule, and manage social media posts via Typefully for Twitter/X, LinkedIn, Threads, Bluesky, Mastodon |
| **video-downloader** | Download YouTube videos with customizable quality and format options |
| **youtube-transcript** | Download YouTube video transcripts, captions, and subtitles |
