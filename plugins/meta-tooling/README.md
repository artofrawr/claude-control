# meta-tooling

Tooling for managing Claude Code plugins, skills, and workflows.

**Version:** 1.0.0

## Skills

| Skill | Description |
|-------|-------------|
| **skill-creator** | Create new skills, modify and improve existing skills, and measure skill performance with evals and benchmarks |

## Usage

Use the skill-creator when building new skills from scratch, editing existing skills, running evals to test skill quality, or benchmarking skill performance.
