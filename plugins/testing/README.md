# testing

Testing skills: test strategies, frameworks, and quality assurance patterns.

**Version:** 1.0.0

## Skills

| Skill | Description |
|-------|-------------|
| **llm-testing** | Comprehensive testing prompts and wordlists for evaluating LLM security, safety, and robustness |
| **playwright-testing** | E2E testing with Playwright - Page Objects, cross-browser, CI/CD |
| **ui-testing** | Visual testing - catch invisible buttons, broken layouts, contrast |

## Usage

Skills are loaded automatically when Claude detects testing-related context. Mention E2E testing, visual testing, LLM security testing, or Playwright to trigger the appropriate skill.
