# writing

Writing skills: copywriting, documentation, editing, and content creation.

**Version:** 1.0.0

## Skills

| Skill | Description |
|-------|-------------|
| **article-extractor** | Extract clean article content from URLs without ads, navigation, or clutter |
| **autoresearch** | Orchestrates end-to-end autonomous AI research projects using a two-loop architecture |
| **avoid-ai-writing** | Audit and rewrite content to remove AI writing patterns and AI-isms |
| **content-research-writer** | Assists in writing high-quality content by conducting research, adding citations, and providing real-time feedback |
| **deep-research** | Universal deep research agent team - 13-agent pipeline for rigorous academic research on any topic |
| **docx** | Create, read, edit, and manipulate Word documents (.docx files) with professional formatting |
| **internal-comms** | Write internal communications - status reports, leadership updates, newsletters, FAQs, incident reports |
| **pdf** | Read, extract, combine, split, rotate, watermark, create, fill, encrypt, and OCR PDF files |
| **pptx** | Create, read, edit, and manipulate PowerPoint presentations (.pptx files) |
| **writing-plans** | Use when you have a spec or requirements for a multi-step task, before touching code |
| **writing-skills** | Use when creating new skills, editing existing skills, or verifying skills work before deployment |
| **xlsx** | Create, read, edit, and manipulate spreadsheet files (.xlsx, .xlsm, .csv, .tsv) |

## Usage

Skills are loaded automatically when Claude detects writing-related context. Mention PDFs, Word docs, spreadsheets, presentations, research, content writing, or AI writing audit to trigger the appropriate skill.
